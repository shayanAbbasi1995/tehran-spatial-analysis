# Tehran Spatial Analysis

Interactive maps and incident visualizations for Tehran districts, built with [Folium](https://python-visualization.github.io/folium/) (Leaflet.js). Created as part of infrastructure diagnostics work for [United For Iran](https://united4iran.org/).

## Maps

| File | Description |
|---|---|
| `tehran_map.html` | General overview map of Tehran |
| `tehran_area_map.html` | Choropleth map of Tehran areas |
| `tehran_districts_map.html` | Choropleth map of Tehran administrative districts |
| `tehran_daytime_distribution.html` | Distribution of incidents by time of day |
| `tehran_weekly_incidents_plot.html` | Weekly incident frequency plot |

Open `index.html` in a browser for a navigation page linking all visualizations.
